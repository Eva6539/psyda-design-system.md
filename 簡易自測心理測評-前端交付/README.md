## 重點

- **識別色**：暖珊瑚 `#F5806C`；中性底柔霧粉 `#F6EBEE`（非識別色）。
- **材質**：黏土（立體物件）× 液態玻璃（卡片表面）。
- **字體**：Noto Sans SC / PingFang；文字錨點 `#4A3340`，不用純黑。
- **動畫**：心情球輕柔跳動、心理電量水波填充、答題聲波/倒數環、生成敘事進度；全部加 `prefers-reduced-motion`。
- **資料**：測評 1 題語音、五項指標(1–10)、心理電量 `(50−總分)/50×100`、任一≥8 帶可點「建議尋求專業支持」、報告對象含長者(質性標籤＋放大字級＋語音播報)。

## 怎麼看

- 工程師：先讀 `前端交付說明.md`，再對照三個 `.html` 原型與 `tokens.css`。
- 原型用本機伺服器開啟體驗最完整（引用 Google Fonts 與相對資源）：
  ```bash
  python3 -m http.server 8000
  # 例：http://localhost:8000/首頁-符合規範.html
  #     http://localhost:8000/測評流程-UX優化版.html
  #     http://localhost:8000/設計規範-識別色與UI.html
  ```
