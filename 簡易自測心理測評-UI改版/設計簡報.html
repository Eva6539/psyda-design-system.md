一套統一的視覺語言

報告 · 核心視覺

心理能量＝
Mochi 抱的愛心

  • Mochi 抱著一顆半滿的愛心，填充比例＝測評百分比（示意 50%）
  • 為什麼用愛心不用電池：保留「充電 / 能量」概念，但更溫暖有情感、不像冷硬器物
  • 等級用詞柔化（例：能量過半），避免「沒電 / 即將關機」的負面感
報告 · 心理能量11
報告 · 五項情緒指標

精緻的情緒側寫，
不是儀表板

  • 心情低落 / 情緒活躍 / 擔憂緊張 / 情緒敏感 / 身心疲憊，各 1–10 分取代原本的「人格特質分析」
  • 襯線大數字 + 質性標籤（偏高 / 中等 / 平穩） + 細長條 + 細線分隔
  • 為什麼：用「偏高 / 平穩」這種人話，比裸數字更好懂、更溫柔
報告 · 五指標12
報告 · 調節建議與療愈

給方向、給安全網，
但不嚇人

  • 調節建議：50 字內、針對最高 1–2 項、具體可執行、支持非診斷
  • 任一指標 ≥8 → 顯示「建議尋求專業支持」，用柔和引述式提醒，非生硬警示框
  • 專屬療愈（Mochi 想對你說）：第二人稱、溫暖、不提數字、不診斷
