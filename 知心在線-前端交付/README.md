# 知心在線 · 前台 UI 交付包（嫩芽花園・暖療癒版）

給工程師的設計交付資料。原型為**自帶圖的單一 HTML**（圖片已內嵌 base64，雙擊即可開），實作時請改用 `assets/` 裡的原始素材。

## 三、情緒系統規格（兩套，各自獨立）

### A. 心理電量（首頁頂部）→ 驅動吉祥物全身立繪 + 場景
資料來源：**最近一次「自測心理測評」** 的電量百分比。
> 電量% 公式（沿用）：`電量% =（50 − 五項總分）/ 50 × 100`

電量% 自動對應 5 種情緒。情緒由四個元素同步表現（**底色維持暖綠識別，不整片改色**）：
**① 吉祥物立繪＋動作 ② 吉祥物背後情緒光暈 ③ 花園枯榮 ④ 能量條顏色**。

| 電量% | 情緒 | 立繪 | 動作 | 情緒光暈 | 花園 | 能量條色 |
|---|---|---|---|---|---|---|
| 0–18 | 哀 | `哀-sad.png` | 啜泣 + 流淚 | 冷藍 | 草重度枯萎、下雨 | 藍 `#6F8CC8` |
| 18–38 | 怒 | `怒-angry.png` | 微抖 + 冒煙 | 橙紅 | 草枯黃焦枯 | 橙 `#E0784F` |
| 38–58 | 普通 | `普通-neutral.png` | 平靜呼吸 | 淡灰 | 偏淡、花少 | 灰綠 `#8F9D88` |
| 58–80 | 喜 | `喜-happy.png` | 微笑、輕浮動 | 綠 | 蔥綠、花、蝴蝶 | 綠 `#34C4BD` |
| 80–100 | 樂 | `樂-joy.png` | 舉手歡呼、Q彈 + 愛心 | 金陽 | 鮮亮繁盛 | 金 `#F5A623` |

- **情緒光暈**＝吉祥物背後的柔光圓暈（localized，不可放右上角以免擋到 orb 按鈕）；天氣只保留「雨」且只下在吉祥物上方。
- 首頁顯示為**靜態結果**（如 76% → 喜）；`心理電量-互動規格.html` 的滑桿僅為示範，呈現完整對應邏輯。
- 電量區排版：第一行「心理电量 · 日期」+ 右側百分比；能量條；下方一行簡要描述（放大、情緒色）。整塊寬度上限約 `162px`，**不可蓋到右側吉祥物**。

### B. 心情打卡（每日）→ 5 點心情量表
| 很好 | 不错 | 普通 | 低落 | 想哭 |
|---|---|---|---|---|
| `很好.png` | `不错.png` | `普通.png` | `低落.png` | `想哭.png` |

- 點選 → 上方吉祥物即時做出對應反應（想哭=哭、很好=愛心…）；選中態放大 + 情緒色光暈；按鈕即時回饋並記錄。
- 與 A（喜怒哀樂）**各自獨立**，不共用。

## 四、動畫規格

| 元件 | 動畫 |
|---|---|
| 吉祥物（idle） | 呼吸上下浮動 `bob` |
| 哀 | `sob`（啜泣抖動）+ 淚滴粒子 `fall` |
| 怒 | `fume`（微抖）+ 蒸氣粒子上升 |
| 樂 / 很好 | `boing`（Q彈擠壓）+ 愛心上飄 |
| 花園 | 草叢 `sway`（相位錯開）、蝴蝶 `flyPath` + 翅膀 `flap` |
| 能量條 | 流動光效 `shine` |

- **全部動畫都包 `@media (prefers-reduced-motion: reduce)`**，使用者關閉動畫時自動靜止。
- 進階動作（揮手 / 眨眼 / 嘴型）**做不到**：此立繪為單張 3D 算圖，毛絨手臂與身體無乾淨分界。請美術提供 **PSD 分層** 或 **Lottie / Rive 骨架檔**，前端即可接入絲滑動畫。

## 五、實作注意

1. **原型 vs 實作**：`prototypes/` 的圖是內嵌 base64（方便預覽），實作請用 `assets/` 的原始檔。
2. **吉祥物定位**：立繪以「腳底」對齊草地基準線；首頁吉祥物窩在花園場景中（草地土堆 + 接觸陰影 + 前排草包覆）。
3. **「普通-neutral.png」為暫代版**（由「喜」改平嘴），建議美術補一張正式的「普通」全身立繪，風格才完全統一。
4. 字體請引入 Noto Sans SC；繪製玻璃卡注意 `backdrop-filter` 的降級。
